AlgebraicInference.GaussianRelation — Type
GaussianRelation(logdensity)

The Gaussian relation determined by quadratic bifunction logdensity.

A Gaussian relation is a equivalence class of quadratic bifunctions. Bifunctions $F_1$ and $F_2$ are equivalent if $F_1 = F_2 + \alpha$ for some $\alpha \in \mathbb{R}$.

If $F: m \to n$ and $F \neq -\infty$, then the Gaussian relation determined by $F$ corresponds to an extended Gaussian distribution on $\mathbb{R}^{m + n}$.

StatsAPI.params — Function
params(d::GaussianRelation)

Represent a Gaussian relation as an extended Gaussian distribution.

Returns a quadruple $(Q, a, B, b)$. If $b = 0$, then $d$ corresponds to the extended Gaussian distribution

\[ \mathcal{N}(a, Q) + \text{null } B.\]

Otherwise, $d$ does not correspond to an extended Gaussian distribution.

source
*(M::Union{UniformScaling, AbstractMatrix}, f::QuadraticFunction)

Compute the image of $f$ under $M$, given by

\[(Mf)(y) = \inf \{ f(x) \mid y = Mx \}.\]

Returns a quintuple $(Q, a, \alpha, B, b).$ If $b \neq 0$, then $Mf = -\infty$. Otherwise,

\[(Mf)(y) = \begin{cases} \langle y, \frac{1}{2}Qy + a \rangle + \alpha & By = 0 \\ \infty & \text{else} \end{cases}.\]

source
Base.adjoint — Function
adjoint(F::QuadraticBifunction)

Compute the adjoint of $F$, given by

\[F^*(y^*, x^*) = \sup \{ F(x, y) - \langle y, y^* \rangle + \langle x, x^* \rangle \mid x, y \}.\]

Returns a quintuple $(Q, a, \alpha, B, b).$ If $b \neq 0$, then $F^* = -\infty$. Otherwise,

\[(F^*)(y, x) = \begin{cases} \langle (y, x), \frac{1}{2}Q(y, x) + a \rangle + \alpha & By = 0 \\ \infty & \text{else} \end{cases}.\]
