GaussianSystem{T₁, T₂, T₃, T₄}(P, S, p, s, σ) where {
    T₁ <: AbstractMatrix,
    T₂ <: AbstractMatrix,
    T₃ <: AbstractVector,

Construct a Gaussian system by specifying its energy function.

You should set σ equal to $s^\mathsf{T} S^+ s$, where $S^+$ is the Moore-Penrose psuedoinverse of $S$.

Solvers

AlgebraicInference.InferenceSolver — Type
InferenceSolver{T₁, T₂ <: Union{Nothing, AbstractVector}}

This is the type constructed by init(ip::InferenceProblem). Use it with solve or solve! to solve inference problems.

An InferenceSolver can be reused to answer multiple queries:

is = init(ip)
sol1 = solve(is)
is.query = query2
sol2 = solve(is)
