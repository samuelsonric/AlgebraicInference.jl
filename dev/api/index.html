AlgebraicInference.InferenceProblem — Type
InferenceProblem{T₁, T₂, T₃, T₄}

An inference problem computes the conditional distribution of $X$ given $Y = e$, where $X$ and $Y$ are random variables whose joint probability distribution is specified by a graphical model.

Solvers

AlgebraicInference.InferenceSolver — Type
InferenceSolver{T₁, T₂, T₃}

A solver for an inference problem.

An InferenceSolver can be reused to answer multiple queries:

is = init(ip)
sol₁ = solve(is)
is.query = query
sol₂ = solve(is)
