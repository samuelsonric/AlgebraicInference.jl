AlgebraicInference.GaussianRelation — Type
GaussianRelation(rf)

A Gaussian relation is a morphism in the hypergraph category $\text{Cond}(\text{GaussEx})$. Internally, it is represented as an equivalence class of partial quadratic bifunctions, where

\[F_1 \sim F_2 \iff F_1 = F_2 + \alpha \text{ for some } \alpha \in \mathbb{R}.\]

struct GaussianRelation{T₁, T₂, T₃, T₄}
    rf::QuadraticBifunction{T₁, T₂, T₃, T₄}
end

Every Gaussian relation $d: m \to n$ is either an extended Gaussian distribution on $\mathbb{R}^{m + n}$ or a failure state $\bot_{m + n}$.

References:

  • Stein & Staton (2021), "Compositional Semantics for Probabilistic Programs with Exact Conditioning" (arXiv:2101.11351)
  • Stein (2022), "Decorated Linear Relations: Extending Gaussian Probability with Uninformative Priors" (arXiv:2204.14024)
  • Stein (2022), "A Hypergraph Category for Exact Gaussian Inference", (https://msp.cis.strath.ac.uk/act2022/papers/ACT2022paper3601.pdf)

AlgebraicInference.QuadraticBifunction — Type
QuadraticBifunction(L, R, f)

A partial quadratic bifunction $F: m \to n$ is a convex function on $\mathbb{R}^{m + n}$ of the form

\[F(x, y) = \begin{cases} \langle (x, y), Q(x, y) + a \rangle + \alpha & B(x, y) = b \\ \infty & \text{else} \end{cases},\]

where $Q$ is positive semidefinite. Internally, $F$ is represented by a triple $(L, R, f)$ such that

\[F(x, y) = f^*(Ry - Lx),\]

where $f^*$ is the convex conjuate of $f$.

struct QuadraticBifunction{T₁ <: AbstractMatrix, T₂ <: AbstractMatrix, T₃, T₄}
    L::T₁
    R::T₂
    f::QuadraticFunction{T₃, T₄}
end

*(M::Union{UniformScaling, AbstractMatrix}, f::QuadraticFunction)

Compute the image of f under M, given by

\[(Mf)(y) = \inf \{ f(x) \mid y = Mx \}.\]

Returns a quintuple $(Q, a, \alpha, B, b).$ If $b \neq 0$, then $Mf = -\infty$. Otherwise,

\[(Mf)(y) = \begin{cases} \langle y, \frac{1}{2}Qy + a \rangle + \alpha & By = 0 \\ \infty & \text{else} \end{cases}.\]

source
AlgebraicInference.conjugate — Function
conjugate(F::QuadraticBifunction)

Compute the convex conjugate of F, given by

\[F^*(x^*, y^*) = \sup \{ \langle y, y^* \rangle - \langle x, x^* \rangle - F(x, y) \mid x, y \}.\]

Returns a quintuple $(Q, a, \alpha, B, b).$ If $b \neq 0$, then $F^* = -\infty$. Otherwise,

\[F^*(x^*, y^*) = \begin{cases} \langle (x^*, y^*), \frac{1}{2}Q(x^*, y^*) + a \rangle + \alpha & By^* = 0 \\ \infty & \text{else} \end{cases}.\]
