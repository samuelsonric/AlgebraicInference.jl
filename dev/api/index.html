StatsAPI.params — Function
params(d::GaussianRelation)

Represent a Gaussian relation $d: m \to n$ as an extended Gaussian distribution $\mathcal{N}(\mu, \Sigma) + \text{null } D$ on $\mathbb{R}^{m + n}$.

Returns a five-tuple $(Σ, μ, D, B, b)$. If $b \notin \text{col } B$, then $d$ is equal to the failure morphism $\bot: m \to n$.
